Новая Палитра

Тестовая страница на основе арта. Волны движутся сверху вниз, звёзды повторяют цвета акцентов. Эффект стекла реализован через прозрачность + тени (без blur).

📄 Текстовые блоки

Золотой акцент используется для заголовков и важных элементов. Циан и розовый — для интерактивных состояний и предупреждений. Тёмный фон сохраняет контраст и не утомляет глаза при долгой работе.

Все элементы используют CSS-переменные. Чтобы сменить тему, достаточно поменять значения в :root.
